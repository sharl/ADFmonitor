# ADFmonitor

現在のアストルティア防衛軍の襲撃兵団をお知らせ

![](Assets/image.png)

メタルーキー軍団の大行進スケジュール・天獄・源世庫パニガルム・昏冥庫パニガルムも確認できるようになりました

## Run

```
git clone https://github.com/sharl/ADFmonitor.git
cd ADFmonitor
pip install -r requirements.txt
python ADFmonitor.py
```
